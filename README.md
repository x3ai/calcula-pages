# Calcula Public Pages

This directory contains local static pages prepared for App Store Connect URLs.

- `support/index.html`: public support page.
- `privacy/index.html`: public privacy policy page.
- `assets/site.css`: shared styling.

Published targets:

- Support: `https://x3ai.github.io/calcula-pages/support/`
- Privacy Policy: `https://x3ai.github.io/calcula-pages/privacy/`

Deploy only the contents of this directory to the `gh-pages` branch. Do not use the repository `docs/` directory as the GitHub Pages source, because it contains internal release materials.
