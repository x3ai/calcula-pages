English

Enable the Keyboard

Open iOS Settings, add Calcula as a keyboard, then press and hold the globe key in any text field to switch to Calcula.

Insert a Result

Type an expression with the Calcula keyboard and tap = to insert the result into the current text field.

Scientific Functions

Tap f(x) on the keyboard header to show scientific functions such as sin, cos, log, powers, and roots.

Settings

Open the Calcula app to adjust decimal precision and thousands separator preferences.

简体中文

启用键盘

打开 iOS 设置，添加 Calcula 键盘，然后在任意输入框长按地球键切换到 Calcula。

插入结果

使用 Calcula 键盘输入表达式，点击 = 即可把结果插入当前输入框。

科学函数

点击键盘顶部的 f(x)，可展开 sin、cos、log、幂运算和根号等科学计算按键。

设置

打开 Calcula App，可调整小数精度和千位分隔等计算偏好。

日本語

iOSの設定を開いてCalculaをキーボードに追加し、任意の入力欄で地球儀キーを長押ししてCalculaに切り替えます。

結果を挿入する

Calculaキーボードで式を入力し、=をタップすると結果が現在の入力欄に挿入されます。

科学関数

キーボード上部のf(x)をタップすると、sin、cos、log、べき乗、平方根などの科学計算キーを表示できます。

한국어

Español

Activar el teclado

Insertar un resultado

Escribe una expresión con el teclado Calcula y toca = para insertar el resultado en el campo de texto actual.

Deutsch

Tastatur aktivieren

Öffne die iOS-Einstellungen, füge Calcula als Tastatur hinzu und halte in einem Textfeld die Globus-Taste gedrückt, um zu Calcula zu wechseln.

Ergebnis einfügen

Gib mit der Calcula-Tastatur einen Ausdruck ein und tippe auf =, um das Ergebnis in das aktuelle Textfeld einzufügen.

Wissenschaftliche Funktionen

Tippe in der Tastaturleiste auf f(x), um wissenschaftliche Funktionen wie sin, cos, log, Potenzen und Wurzeln anzuzeigen.

Einstellungen
